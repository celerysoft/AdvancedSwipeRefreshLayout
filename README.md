# AdvancedSwipeRefreshLayout

## Features

* Pull to refresh

* Push to load more

* Custom header view

* Custom footer view

* Work as same as origin SwpieRefreshLayout, easy to get start

## Technical Information

* Required minimum API level: 15(Android 4.0.4)

* Supports all the screens sizes and density.
